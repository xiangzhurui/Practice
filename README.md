# PracticeJava

> 此项目主要用作放置Demo、Java 编程学习验证，以及一些工具类，方法，框架配置等。

## 运行环境与开发工具

* JDK 1.8
* Eclipse Neon (4.6.1)
* Atom
* Maven 3.3
* Tomcat 8.0

## 基础框架

* Spring 4.3
* Spring MVC 4.3
* MyBatis 3.4
* mybatis-spring 1.3
* SLF4J + Log4j 2

## 数据库

* MySQL Community Server 5.7

## 前端框架或库

* Bootstrap 3
* jQuery 3
